WHAT IT DOES
[SYSTEM_OVERVIEW]

BankAPI is a fully secured REST API that handles user authentication and bank account management. Users register and receive a JWT token on login. That token is required to access all account endpoints — creating accounts, checking balances, depositing, and withdrawing.

The entire system runs in Docker containers — Spring Boot app and PostgreSQL database — deployed on an AWS EC2 instance accessible from anywhere in the world.

API ENDPOINTS // FULL REFERENCE
POST /auth/register

Register a new user. Returns 201 Created. Validates username and password — rejects duplicates and weak passwords.

PUBLIC — NO TOKEN REQUIRED

POST /auth/login

Login with credentials. Returns structured JSON with JWT token: {"token": "eyJ..."}

PUBLIC — NO TOKEN REQUIRED

POST /accounts

Create a new bank account with owner name and initial balance. Returns full account details.

PROTECTED — JWT REQUIRED

GET /accounts/{id}

Retrieve account details by ID. Returns 404 if account not found.

PROTECTED — JWT REQUIRED

GET /accounts/{id}/balance

Check current balance for a specific account.

PROTECTED — JWT REQUIRED

POST /accounts/{id}/deposit

Deposit funds into an account. Rejects negative amounts.

PROTECTED — JWT REQUIRED

POST /accounts/{id}/withdraw

Withdraw funds. Returns 400 if insufficient balance.

PROTECTED — JWT REQUIRED

TECH STACK // HOW IT'S BUILT
[LAYER_01 — LANGUAGE & FRAMEWORK]

Java 21 + Spring Boot

Core language and framework. Spring Boot handles the REST layer, dependency injection, and JPA integration.

[LAYER_02 — DATABASE]

PostgreSQL + Spring Data JPA

Persistent storage for users and bank accounts. JPA handles ORM — entities map directly to database tables.

[LAYER_03 — SECURITY]

Spring Security + JWT + BCrypt

Passwords hashed with BCrypt. JWT tokens generated on login and validated on every protected request via a custom filter chain.

[LAYER_04 — INFRASTRUCTURE]

Docker + AWS EC2

App and database containerized with Docker Compose. Deployed on an Ubuntu EC2 instance. Publicly accessible at port 8080.
